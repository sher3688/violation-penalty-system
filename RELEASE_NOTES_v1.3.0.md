# 違規罰單系統 v1.3.0

## 新增：月份封存 ZIP

- 月份匯出頁新增「下載封存 ZIP」。
- ZIP 內含：
  - 當月全部有效罰單 PDF。
  - 當月全部有效罰單 Excel。
  - 封存說明 JSON（月份、產生時間、筆數、金額、通知單編號）。
- ZIP 使用壓縮模式，降低本機備份檔案體積。

## 刪除安全強化

- 當月未先產生封存 ZIP 時，無法勾選備份確認。
- 必須確認 ZIP 已下載且可正常開啟，才能啟用月份刪除。
- 切換月份後，封存確認狀態自動重設，避免誤刪其他月份。
- 刪除完成後，封存確認狀態自動清除。

## 既有功能保留

- 每張罰單最多兩張照片。
- 刪除案件後，Excel、PDF、統計不再出現殘留資料。
- 月份刪除同步清除罰單、照片、繳款及申訴關聯資料。
